# Roadmap

This document outlines the goals and next steps for the Awesome UniFi project.

## Current Status

- **Repository**: Live and accepting contributions
- **Awesome List Submission**: PR pending to [sindresorhus/awesome](https://github.com/sindresorhus/awesome)
- **Linting**: Passes `awesome-lint` via GitHub Actions
- **Content**: 200+ tools across 20+ categories

## Goals

### Short-term (Q1-Q2 2026)

#### 1. Get Accepted to Main Awesome List
- [ ] Monitor and respond to PR feedback from sindresorhus/awesome maintainers
- [ ] Address any requested changes promptly
- [ ] Ensure continued compliance with awesome-lint
- [ ] Update badge URL once accepted

#### 2. Content Expansion
- [ ] Add UniFi Network 9.x compatible tools as they emerge
- [ ] Expand MCP Servers section (growing ecosystem)
- [ ] Add AI/ML tools for network analytics
- [ ] Include more international/localized tools
- [ ] Add UniFi Site Manager tools

#### 3. Quality Improvements
- [ ] Add star counts to all entries
- [ ] Mark unmaintained projects clearly
- [ ] Verify all links are active
- [ ] Add "Last Updated" indicator to entries

### Medium-term (Q3-Q4 2026)

#### 4. Community Building
- [ ] Promote on r/Ubiquiti and UniFi Community forums
- [ ] Reach out to tool maintainers for cross-promotion
- [ ] Consider creating a Discord/Matrix for contributors
- [ ] Add CODEOWNERS file for faster PR reviews

#### 5. Enhanced Automation
- [ ] Add automated link checking via GitHub Actions
- [ ] Implement star count update workflow
- [ ] Add stale project detection

#### 6. Content Organization
- [ ] Consider splitting into sub-pages if list grows too large
- [ ] Add a "New Additions" section for visibility
- [ ] Create category-specific contribution templates

### Long-term (2027+)

#### 7. Ecosystem Integration
- [ ] Partner with Ubiquiti Community organization
- [ ] Create a companion website with search/filter
- [ ] Add project comparison tables for similar tools

## Maintenance Plan

### Weekly
- Review and merge PRs
- Respond to issues

### Monthly
- Check for broken links
- Update star counts for top 50 projects
- Remove defunct projects

### Quarterly
- Full link audit
- Review category organization
- Update this roadmap
